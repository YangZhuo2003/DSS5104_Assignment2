# Fraud and Anomaly Detection with Deep Learning

## Setup
Install required dependencies:

pip install -r requirements.txt

---

## Run Experiments
If using scripts:

python main.py

If using notebooks, run them in sequence as listed below.

---

## Notebooks
- 01_feature_engineering.ipynb  
- 02_classical_models.ipynb  
- 03_smote_models.ipynb  
- 04_deep_learning.ipynb  
- 05_cost_analysis.ipynb  

---

## Outputs
Running the notebooks/scripts will generate:
- Model performance tables (AUPRC, ROC AUC, cost)
- Precision–Recall curves
- Cost vs threshold plots
- Feature engineering comparison results

---

## Notes
- Datasets are not included due to size constraints  
- Please download IEEE-CIS and PaySim datasets separately  
- All results in the report can be reproduced from the notebooks
