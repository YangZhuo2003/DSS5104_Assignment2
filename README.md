# Fraud and Anomaly Detection with Deep Learning

## Overview
This project investigates the effectiveness of classical machine learning and deep learning methods for fraud detection on imbalanced datasets.

We evaluate models on:
- **IEEE-CIS Fraud Detection (primary dataset)**
- **PaySim (secondary dataset)**

The focus is on:
- Handling extreme class imbalance
- Comparing classical vs deep learning approaches
- Evaluating models under realistic cost constraints

## Evaluation

Models are evaluated using:

- **AUPRC (primary metric)**
- ROC AUC
- Cost-sensitive evaluation

### Cost Function
Total Cost = FN × 500 + FP × 2

Precision, Recall, and F1 are reported at the **cost-optimal threshold**, ensuring consistency with business objectives.
